# pi-k2

**k3-2o's personal pi coding agent config** — extensions, themes, and skills bundled as a pi package.

## What's included

### Extensions
| File | Description |
|------|-------------|
| `ask-user-question.ts` | Ask the user multiple-choice questions |
| `auto-commit-on-exit.ts` | Auto-commit git changes on exit |
| `btw.ts` | Side-conversation hover window — talk to the agent while pi is running |
| `clipboard.ts` | Copy text to system clipboard |
| `multi-edit.ts` | Multi-file surgical edits |
| `omnisearch-gateway.ts` | Web search, discovery, and extraction |

### Themes
- `catppuccin-mocha.json` — Catppuccin Mocha
- `dracula.json` — Dracula
- `my-catppuccin-mocha.json` — Custom Catppuccin Mocha variant
- `nightowl.json` — Night Owl

### Skills
- `skill-creator/` — Create, refactor, and productionize Agent Skills

## Installation

```bash
pi install git:github.com/k3-2o/pi-k2
```

Also install the standalone packages:

```bash
pi install git:github.com/k3-2o/pi-read-image
pi install git:github.com/k3-2o/pi-repo-baby
```
